Basic Lane Detection

ABSTRACT

Many has done this basic lane detection, this is the start project for self-driving car. This would find lanes on the test_image and test_video and marked the lanes on to the image and the video.

Study Steps

1. Load road image

2. Convert to GrayScale

3. Apply Gaussian Blur




4. Apply Canny Edge Detection Operator

5. apply bitwise on Region Of Interest to extract lines

6. marked the line on the image
